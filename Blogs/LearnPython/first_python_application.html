First Python Application

Here is a simple Python application that generates a random horoscope based on the user's astrological sign: 

                
                    import random
                    
                    # List of horoscopes
                    horoscopes = [
                        "You will have a great day today!",
                        "Expect some unexpected surprises today.",
                        "Today is a good day to take risks and try new things.",
                        "Today is a good day to focus on your relationships.",
                        "Today is a good day to focus on your personal growth.",
                    ]
                    
                    # Dictionary of astrological signs and corresponding horoscopes
                    signs = {
                        "Aries": horoscopes[0],
                        "Taurus": horoscopes[1],
                        "Gemini": horoscopes[2],
                        "Cancer": horoscopes[3],
                        "Leo": horoscopes[4],
                    }
                    
                    # Get the user's astrological sign
                    sign = input("Enter your astrological sign: ")
                    
                    # If the sign is in the dictionary, print the corresponding horoscope
                    if sign in signs:
                        print(signs[sign])
                    
                    # If the sign is not in the dictionary, print a random horoscope
                    else:
                        print(random.choice(horoscopes))

This program starts by importing the random module, which will be used to choose a random horoscope if the user's astrological sign is not in the dictionary. The program then defines a list of horoscopes and a dictionary of astrological signs and corresponding horoscopes.

The program prompts the user to enter their astrological sign, and then checks if the sign is in the dictionary. If it is, the program prints the corresponding horoscope. If the sign is not in the dictionary, the program uses the random.choice() function to choose a random horoscope from the list and print it.
